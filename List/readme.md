# List

## architecture
| name| introduction |
| --- | --- |
| listSize | 元素个数 |
| pos | 当前位置 |
| length | 返回元素个数 |
| currPos | 返回列表当前位置 |
| currentElement | 返回当前位置元素 |
| clear( ) | 清空列表 |
| toString( ) | 返回列表字符串 |
| find( ) | 查找元素索引 |
| contain( ) | 查找元素是否存在 |
| insert( ) | 现有的元素后插入新元素 |
| append( ) | 在末尾添加新元素 |
| remove( ) | 删除元素 |
| front( ) | 当前位置移动到第一个元素 |
| end( ) | 当前位置移动到最后一个元素 |
| prev( ) | 当前位置前移一位 |
| next( ) | 当前位置后移一位 |
| moveTo( ) | 将当前位置移动到指定位置 |
