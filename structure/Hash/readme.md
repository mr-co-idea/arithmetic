## 观点
* 根据数据自身通过公共函数产生一个比数组长度小的键值，进而实现快速存入和查找
* 自身 ---> 键值 ---> 散列化 ---> 位置数值
## features
* 删除和取用操作非常快
* 查询操作效率低下
* HashTable
