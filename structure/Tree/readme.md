## 遍历
* 中序遍历
   * 升序访问树中所有节点，先访问左子树，再访问 根节点，最后访问右子树
* 先序遍历
   * 先访问根节点，然后以同样方式 访问左子树和右子树
* 后序遍历
   * 先访问叶子节点，从左子树到右子树，再到根节点
